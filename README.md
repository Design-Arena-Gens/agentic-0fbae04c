## Carbon Footprint Reduction PDF Generator

Generate a professional PDF proposal for a project to reduce carbon footprint (empreinte carbone).

### Local development

1) Install dependencies:
```bash
npm install
```
2) Start dev server:
```bash
npm run dev
```
3) Open http://localhost:3000

### Build
```bash
npm run build
```

### Deploy (Vercel)
```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-0fbae04c
```
